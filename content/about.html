AdBlock Classic

AdBlock Classic is an open-source extension for K-Meleon. It is an updated and de-branded descendant of the classic AdBlock Plus extension for Firefox. It includes the improvements from Hermes' port of ABPrime, and it uses a custom interface for K-Meleon. AdBlock Classic supports AdBlock Plus filters and rules. The extension is open-source software available under the MPL 2.0. It is distributed freely, as-is, and without warranty. That said, you're welcome to visit the K-Meleon forums if you have questions or issues.

Updates

AdBlock Classic will only update when you ask it to; it will not download or install updates in the background. To update the extension itself, visit the extension's page and click Install AdBlock Classic. K-Meleon will install the new version while preserving your settings or blocking rules.

To update your blocking rules:

  1. In AdBlock Classic's menu, click Settings and the filter settings dialog will pop up. On the Filter subscriptions tab, you'll see the various lists of blocking rules that you are using.
  2. Click the Actions button beside the list you want to update and select Update filters in the menu. The extension will download new blocking rules from the subscription URL.

Upgrading

If you have AdBlock Classic 1 or ABPrime installed, disable them from about:addons to continue using their settings and rules list. You can use your rules from a previous installation of AdBlock Plus but not the settings.

Custom rules to allow ads on a site

  1. Copy the domain name only from the address bar. For example, on the page "https://en.wikipedia.org/wiki/Advertising" you would copy only "wikipedia.org" from the address. Paste it into the template in the next step.
  2. Replace the "example.com" with your domain name: @@||example.com^$document and copy it to the clipboard.
  3. In AdBlock Classic's menu, click Settings and the filter settings dialog will pop up. Select the Custom filters tab and click the Add filter button.
  4. Paste your blocking exception, and AdBlock Classic will no longer block ads on that site.

Custom rules to hide elements

  1. Open the page in K-Meleon.
  2. Press Ctrl + U or go to View - Source Code in the menu. This will open the page's source code in a new window.
  3. Find the "id" of the element that you want to block. It will be in quotation marks and highlighted as blue text.
  4. Copy the id of the element and paste it into the template in the next step.
  5. To create your blocking rule, replace the "id" in ###id and copy the rule to your clipboard.
  6. In AdBlock Classic's menu, click Settings and the filter settings dialog will pop up. Select the Custom filters tab and click the Add filter button.
  7. Paste your blocking rule, and the element should disappear.

You can hide elements by class using ##.something instead of ###something. You can also target your custom rules for only certain websites using AdBlock Plus syntax.
